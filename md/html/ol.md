#ol有序列表标签

|属性|说明|
|:-  |:- |
|compact|规定列表呈现的效果比正常情况更小巧。HTML5 中不支持。HTML 4.01 中不赞成使用。|
|revesed|规定列表顺序为降序。(9,8,7,...)|
|start|规定有序列表的起始值。|
|type|规定在列表中使用的标记类型。(1,a,A,i,I)|

#注释
    1.通常ol是用在有顺序的列表中，其余跟ul基本一致
    2.如果想从自己想要的数字开始排列就在li中加入value，填入想要的数字。
    3.请使用 CSS 来定义列表的类型。

```
<ol>
	<li>要求1</li>
	<li>要求2</li>
	<li>要求3</li>
</ol>
```

#例子
<ol>
	<li>要求1</li>
	<li>要求2</li>
