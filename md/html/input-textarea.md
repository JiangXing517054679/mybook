#input-textarea文本域标签

|属性|值|描述|
|：-|：-|:-|
|cols|number|规定文本的列数|
|rows|number|规定文本的行数|

#注意
            1.没有vlaue属性，它的值是直接写在标签对中间
            2.textarea   表示文本域，专门用于多行文本

```
自我介绍: <textarea name="description" cols="100" rows="10" placeholder="看不到我">66666</textarea><br />
```
